📌 Project Overview

This project analyzes Amazon sales data to uncover key business insights using Microsoft Excel and Power BI. The dashboard provides a clear view of sales performance, order distribution, and operational efficiency.

It demonstrates end-to-end data analysis including data cleaning, transformation, and interactive visualization.# Amazon-Sales-Data-Analysis
I am recently worked on a Sales Data Report &amp; Dashboard in MS-Excel and Power BI, with the goal of going beyond visuals and actually understanding what the data is telling.

🎯 Business Objective

The goal of this project is to:

Monitor overall sales performance
Analyze order status (Shipped, Unshipped, Cancelled)
Identify top-performing product categories
Evaluate delivery and fulfillment efficiency
Understand regional sales distribution

🛠️ Tools & Technologies Used
Microsoft Excel → Data Cleaning & Preprocessing
Power BI → Dashboard & Data Visualization
Power Query → Data Transformation

📂 Dataset Description

The dataset includes the following key attributes:

📅 Date (Order Date)
📦 Category (Kurta, Set, Western, Top, Saree)
🚚 Courier Status (Shipped, Unshipped, Cancelled)
🏢 Fulfillment (Amazon, Merchant)
🚛 Delivery Partner (Amazon, Easy Ship)
📏 Size (S, M, L, XL, XXL, etc.)
🌍 Ship State (Location-wise orders)
💰 Sales Amount

🔄 Data Cleaning & Preparation
Removed inconsistent and null values
Standardized date formats
Cleaned categorical fields (Category, Size, Status)
Performed transformations using Power Query
Created aggregated measures for dashboard KPIs

📊 Key KPIs
💰 Total Sales Amount: 78,593K
📦 Total Orders (based on count metrics)
🚚 Order Status Distribution
📍 State-wise Order Count

📈 Key Insights

📦 Order Status Analysis
~85% orders are successfully shipped
Small percentage of orders are cancelled or unshipped, indicating good operational efficiency

👕 Product Category Performance
Top categories: Set and Kurta dominate sales
Categories like Saree and Top have comparatively lower demand

🚚 Fulfillment Insights
Amazon fulfillment handles majority of orders
Merchant fulfillment contributes less but still significant

🚛 Delivery Partner Analysis
Amazon delivery handles most shipments
Easy Ship is used for a smaller portion

📏 Size Distribution
Most demanded sizes are M, L, and XL
Smaller sizes (XS, S) have lower demand

🌍 Regional Insights
Highest orders come from:
Maharashtra
Karnataka
Tamil Nadu
Telangana
Indicates strong demand in southern and western regions of India
